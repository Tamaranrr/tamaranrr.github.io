Acerca del proyecto

Campaign Email Builder — Stack tecnológico y funcionalidades

🚀

Campaign Email Builder es una plataforma de email marketing asistida por IA, diseñada para las marcas Miles Car Rental (MCR) y Viajemos (VIS). Permite crear, diseñar, programar y monitorear campañas de email, conectado directamente con ActiveCampaign.

🎨 Frontend

React 18 Vite DM Sans CSS Variables
  • SPA con 8 vistas navegables sin recarga
  • Theming dinámico por marca (MCR rojo / VIS azul)
  • Editor visual de secciones de email
  • Preview en tiempo real del HTML del email
  • Responsive y optimizado para equipos de marketing

⚙️ Backend

FastAPI Python 3.11 PostgreSQL Pydantic
  • API REST en puerto 8050 con CORS habilitado
  • PostgreSQL con JSONB para versionado de emails
  • Sincronización automática con ActiveCampaign al inicio
  • Pydantic v2 para validación de esquemas
  • Docker Compose para despliegue (db + backend + frontend)

🤖 Inteligencia Artificial

GPT-4o OpenAI API DALL-E 3
  • Generación de asunto, preheader y cuerpo del email
  • Template Lab: variaciones automáticas de plantillas
  • Recomendación de fechas y horas óptimas de envío
  • Insights basados en historial real de ActiveCampaign
  • Generación de imágenes con DALL-E 3 para emails

📡 ActiveCampaign

AC REST API Webhooks
  • Creación y envío de campañas desde la plataforma
  • Sincronización de historial de campañas (open/click rates)
  • Gestión de biblioteca de imágenes AC
  • Envío de emails de prueba antes del envío masivo
  • Análisis de historial para insights de IA

📧 Editor de Email

12+ secciones HTML generado
  • Banner superior, Hero, Cuerpo de texto
  • 2 columnas con números destacados
  • Galería de imágenes (Grid cards)
  • Testimoniales y citaciones
  • Footer legal con links de desuscripción
  • Historial de versiones por campaña

📁 Estructura del proyecto

Monorepo Docker
  • src/ — Frontend React (views, components, lib)
  • backend/app/ — FastAPI (routes, services, schemas)
  • backend/app/data/ — Persistencia JSON local
  • docker-compose.yml — Orquestación multi-contenedor
  • Puerto frontend: 5180 · Backend: 8050
